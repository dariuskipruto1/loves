# Love Surprise Page for Clare

A beautifully crafted, romantic surprise webpage filled with love, animations, and personalized messages for Clare from Darius.

## How to Use

1. Start a local server: `python3 -m http.server 8000` (or use any local server)
2. Open `http://localhost:8000` in your web browser.
3. Enjoy the automatic romantic music and floating hearts!
4. Click the "Click to Reveal More Love" button to see additional messages, confetti, and heart bursts.

- Personalized romantic messages for Clare
- Stunning gradient background with sparkling effects and pulsing colors
- Animated floating hearts with glow effects
- **20 moving emojis** with unique animation patterns (hearts, flowers, butterflies, stars, sparkles, rainbows)
- **Dynamic random emojis** appearing continuously across the screen
- **Edge decorations** with floating and animated love emojis around the page borders
- Interactive button that triggers confetti and heart burst animations
- Continuous floating heart emojis in the background
- **Background music** with automatic playback (plays when the page loads)
- Responsive design with smooth animations
- Elegant typography and styling
